Simple form

This is a simple Bootstrap form, extended with Angular Form Messages directives.

  • All fields are required
  • Added af-submit to the form for handling the submit callback when the form is valid. The submit callback returns a rejecting promise with 'server side' errors.
  • Added no-validate to the form so that default browser validation is disabled.
  • Added af-field directives on the form elements to extend ngModel
  • Added af-field-state directives on the form groups to show for example a 'has-error' class when the form is invalid
  • Added af-messages divs next to the form elements to show the message
  • Added af-messages div with af-alert attribute for messages that belong to the form, not to a specific field.
  • Added af-submit-button that is disabled while submitting
  • Showing a spinner while submitting
  • Validation is triggered by the default value: 'change'
